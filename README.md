## 📝 기록 방법
각 책을 아래 포맷으로 기록합니다.

1. **《책 제목》 — 저자**
   - 읽은 기간: YYYY.MM.DD ~ YYYY.MM.DD
   - 상태: ✅ 완독 | 📖 읽는 중 | ⏸ 보류
   - 분야/주제: 책의 분야 or 주제
   - 추천 여부: 👍 추천 | 🤔 보통 | 👎 비추천
   - 한줄평: 짧은 느낌 요약
